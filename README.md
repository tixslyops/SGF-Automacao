# Automação de Extração de Relatórios

Automação desenvolvida em Python para gerar, baixar e processar relatórios de um sistema web, convertendo os dados para CSV e organizando os arquivos automaticamente.

## Tecnologias

- Python
- Selenium
- Pandas
- xlrd
- WebDriver Manager
- Logging

## Funcionalidades

- Login automático no sistema.
- Aplicação de filtros de período.
- Geração e exportação de relatórios.
- Monitoramento do download do arquivo.
- Conversão de XLS para CSV.
- Organização automática dos arquivos.
- Registro de logs da execução.
- Tratamento de exceções.
  
## Dependências

```
selenium
pandas
xlrd
webdriver-manager
python-dotenv
```

## Observação

Esta versão foi adaptada para publicação. Credenciais, URLs, caminhos internos e demais informações confidenciais foram removidos, mantendo apenas a lógica da automação.
